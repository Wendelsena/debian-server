# Infra Lab

Laboratório de infraestrutura utilizando Docker, PostgreSQL, Nginx, Prometheus, Grafana e cAdvisor.

## Objetivo

Projeto desenvolvido para estudo de administração Linux, containers Docker, monitoramento e observabilidade.

## Arquitetura

- Debian 13
- Docker
- Docker Compose
- PostgreSQL 17
- pgAdmin
- Nginx
- Prometheus
- Grafana
- Node Exporter
- cAdvisor
